Para importar tickets automáticamente desde tu correo necesitas credenciales de Google Cloud:

  1. Ir a console.cloud.google.com
  2. Crear un proyecto nuevo
  3. Habilitar la Gmail API
  4. Crear una credencial OAuth 2.0 (tipo: aplicación web)
  5. Agregar http://localhost como origen autorizado
  6. Copiar el CLIENT_ID y la API_KEY en los campos abajo

Sincronización con Google Sheets

Los datos (tickets, escalamientos, base de conocimiento y configuración) se guardan también en una planilla de Google Sheets en el Drive de la cuenta conectada, además de en este navegador. Así puedes entrar desde cualquier navegador o equipo con la misma cuenta de Google y ver siempre los mismos datos.

  1. En el mismo proyecto de Google Cloud, habilitar Google Sheets API y Google Drive API
  2. No se necesitan credenciales nuevas — usa el mismo CLIENT_ID y API_KEY de arriba
  3. Al conectar Gmail, el navegador pedirá también permiso para Sheets y Drive
